# ftb_quest:command

Executes each command as the player.

| Name      | Type     | Required | Description |
| --------- | -------- | -------- | ----------- |
| `command` | string[] | yes      |             |

## Example

```json
{
  "type": "ftb_quest:command",
  "command": ["say worked"]
}
```
