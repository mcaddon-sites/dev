# ftb_quest:xp

Grants XP or levels.

| Name        | Type    | Required | Description |
| ----------- | ------- | -------- | ----------- |
| `amount`    | number  | yes      |             |
| `is_levels` | boolean |          |             |

## Example

```json
{
  "type": "ftb_quest:xp",
  "amount": 5,
  "is_levels": true
}
```
