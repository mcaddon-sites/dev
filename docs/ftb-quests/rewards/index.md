# Reward Types

A list of all reward types for quests.

| Reward Type                               | Description                                      |
| ----------------------------------------- | ------------------------------------------------ |
| [`ftb_quest:item`](./item.md)             | Gives an item stack.                             |
| [`ftb_quest:loot_table`](./loot_table.md) | Runs `loot give @s loot <table>` for each entry. |
| [`ftb_quest:command`](./command.md)       | Executes each command as the player.             |
| [`ftb_quest:xp`](./xp.md)                 | Grants XP or levels.                             |
