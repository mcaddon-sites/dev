# ftb_quest:effect_active

Triggers while the named effect is active.

| Name     | Type   | Required | Description |
| -------- | ------ | -------- | ----------- |
| `effect` | string | yes      |             |

## Example

```json
{
  "type": "ftb_quest:effect_active",
  "effect": "minecraft:haste"
}
```
