# ftb_quest:item_use

Counts item uses.

| Name      | Type   | Required | Description |
| --------- | ------ | -------- | ----------- |
| `item_id` | string | yes      |             |

## Example

```json
{
  "type": "ftb_quest:item_use",
  "item_id": "minecraft:copper_sword"
}
```
