# ftb_quest:fall_distance

Passes when fall distance is in range.

| Name                | Type            | Required | Description |
| ------------------- | --------------- | -------- | ----------- |
| `required_distance` | number or range |          |             |

## Example

```json
{
  "type": "ftb_quest:fall_distance",
  "required_distance": 10
}
```
