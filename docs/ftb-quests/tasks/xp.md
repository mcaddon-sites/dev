# ftb_quest:xp

Checks total XP or levels.

| Name          | Type            | Required | Description |
| ------------- | --------------- | -------- | ----------- |
| `required_xp` | number or range | yes      |             |
| `uses_levels` | boolean         |          |             |

## Example

```json
{
  "type": "ftb_quest:xp",
  "required_xp": 10,
  "uses_levels": true
}
```
